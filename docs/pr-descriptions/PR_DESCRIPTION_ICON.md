# アイコンの実装と反映

## 概要

Salmon Run Scenario Hubのブランドアイコンを実装し、ページヘッダー、TOPページ、faviconに反映しました。アイコンはSplatoon 3のサーモンランテーマに基づいたデザインで、ゴールデンエッグ（金イクラ）とシャケ（サーモン）、デジタルネットワークパターンを組み合わせたものです。

## 主な変更内容

### 1. アイコンコンポーネントの作成 (`app/components/LogoIcon.tsx`)

- Next.jsの`Image`コンポーネントを使用した再利用可能なアイコンコンポーネント
- サイズとクラス名をカスタマイズ可能
- `priority`属性で優先読み込みを設定

### 2. ヘッダーへのアイコン追加 (`app/components/layout/Header.tsx`)

- ロゴ部分にアイコンを追加
- レスポンシブ対応（モバイルでも適切に表示）
- ホバー時のトランジション効果を追加

### 3. TOPページへのアイコン追加 (`app/page.tsx`)

- ヒーローセクションの上部に大きなアイコン（80px）を追加
- ドロップシャドウ効果を適用

### 4. Faviconの設定 (`app/layout.tsx`)

- メタデータにアイコン情報を追加
- `icon`と`apple`アイコンとして`/logo.png`を設定

### 5. 画像ファイルの配置

- `public/logo.png`としてアイコン画像を配置
- Next.jsの静的ファイル配信機能を活用

## アイコンのデザイン特徴

- **ゴールデンエッグ（金イクラ）**: 中央やや左寄りに配置された光る球体
- **シャケ（サーモン）**: ゴールデンエッグの右隣に配置された可愛らしい魚
- **デジタルネットワークパターン**: 回路トレースやデータストリームを表現するオレンジ色の線
- **カラーパレット**: オレンジ、イエロー、黒を基調とした暖色系
- **背景**: シンプルな黒背景で、カラフルな要素を際立たせる

## 技術的な詳細

### Next.js Imageコンポーネントの活用

- 自動的な画像最適化
- レスポンシブ画像の生成
- パフォーマンスの向上

### レスポンシブデザイン

- ヘッダー: モバイルでは縦並び、デスクトップでは横並び
- アイコンサイズ: ヘッダー（32px）、TOPページ（80px）で適切に調整
